Skyggen fra P / Aki Palikaras / at være pårørende til psykisk syge. Den beskriver, hvordan det er at være vidne til, at ens familiemedlemmer ændrer personlighed, og hvordan det føles, når man er nødt til at være den, der får sine kære indlagt - også mod deres vilje. Samtidig viser hovedpersonens fortælling, at man trods psykisk sygdom kan styre efter det gode i livet og skabe sig sin egen familie og karriere - og komme ud af skyggen fra de psykiatriske afdelinger. / 32 / 380 / Hvor havde hovedpersonen værnepligt? / Jylland / Bornholm / Luxembourg / København / Hvorfor var naboen bekymret for hovedpersonens mor? / Fordi der ikke var lys i huset / Fordi der var meget stille / Fordi hun ikke havde set hende uden for i et stykke tid / Fordi moren havde trukket alle gardinerne for / Hvad vil hovedpersonen låne af naboen? / En stige / Et brækjern / En mountainbike / Hendes telefon / 2 / 3 / 1
